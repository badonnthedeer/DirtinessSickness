# DirtinessSickness
Adds a moodle representing player hygiene. If dirty for too long, you might get sick!

<h1>Allows for customization in</h1>
<ul>
    <li>Sickness Gain</li>
    <li>Sickness Gain Chance</li>
    <li>Moodle Thresholds</li>
    <li>and More!</li>
</ul>
